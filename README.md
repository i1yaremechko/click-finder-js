# Click Finger Project

Проект для відстеження кліків та аналізу статистики користувачів.

## Технології
* HTML5 / CSS3 (Sass / SCSS)
* JavaScript (Vanilla JS)

## Як запустити локально
1. Відкрийте проект у VS Code.
2. Запустіть компіляцію стилів: `sass --watch styles/index.scss styles.css`
3. Запустіть файл `index.html` через розширення **Live Server**.
